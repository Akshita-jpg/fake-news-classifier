# 📰 Fake News Detection using Deep Learning
![Python](https://img.shields.io/badge/Python-3.10-blue)
![Deep Learning](https://img.shields.io/badge/Model-LSTM-green)

This project builds a Fake News Classifier using NLP and a Bi-directional LSTM model.

---

## 🚀 Features
- Text preprocessing (NLTK, Gensim)
- Stopword removal
- Tokenization & padding
- Word embeddings
- Bi-directional LSTM model
- Model evaluation using accuracy & confusion matrix

---

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- NLTK
- Gensim
- Scikit-learn
- Matplotlib / Seaborn

---

## 📊 Model Performance
- Training Accuracy: ~99%
- Validation Accuracy: ~99%
